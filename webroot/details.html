Rules

Cooldown (in minutes) = Max ( fatigue * 10, 1 + ( distance in lightyears * ( 1 - bonus ) ) )

Fatigue (arbitrary number) = Max ( fatigue, 1 ) * ( 1 + ( distance in lightyears * ( 1 - bonus ) ) )

Decay = 0.1 Fatigue per minute

Be aware that Fatigue can be expressed in minutes by multiplying the fatigue value by 10. But keep in mind that there is no benefit in waiting for the timer to go below 10 minutes.
